Métodos de Ordenação
Insertion Sort
Ordenação por Inserção

Funciona de maneira similar ao modo como se organiza cartas em um baralho. Gradualmente constrói a porção ordenada do array, inserindo cada novo elemento na posição correta.

Selection Sort
Ordenação por Seleção

Simples, mas ineficiente para grandes conjuntos de dados. Repetidamente encontra o menor elemento do array e o move para a posição correta, fazendo isso para todos os elementos.

Quick Sort
Ordenação Rápida

Um algoritmo eficiente que utiliza o conceito de dividir e conquistar. Escolhe um "pivô" e particiona o array em sub-arrays menores e maiores que o pivô, ordenando-os recursivamente.

Merge Sort
Ordenação Por Mesclagem

Um algoritmo de divisão e conquista que classifica um array primeiro dividindo-o em arrays menores e, em seguida, construindo o array novamente da maneira correta para que seja classificado.

Shell Sort
É uma extensão do algoritmo de ordenação por inserção

É o mais eficiente algoritmo de classificação dentre os de complexidade quadrática

Bubble Sort
Ordenação em Bolha

É um algoritmo simples de ordenação que recebe como entrada uma lista de elementos e produz uma lista ordenada de acordo com um critério.
